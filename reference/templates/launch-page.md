# Template — Launch Page (the hub)

One page. It is the single URL you share — the video and deck hang off it. Hero = the hook; one primary CTA (the repo).

## Sections (in order)
1. **Hero** — [mixed-weight headline: one emphasized phrase] + [one-line subhead: the workflow + decision] + [primary CTA → repo].
2. **The three things it does** — [verb + object] ×3, plain language. Decisions, not features.
3. **The visible artifact** — embed the duel, a sample exchange, or the short. Show it working; don't describe it.
4. **Who built it / why** — [the founder line, if the hook or a Gate 4 flag calls for it].
5. **Get it** — repo link, setup in one glance, [judge-guide link if shipped].

## Notes
- Hub rule: the video and deck **embed or link from here**. One URL, not a scatter.
- Build it through the brand's design system (see `reference/tooling.md`), not freehand.
- Signal color ~5%. One emphasized phrase in the hero. No emoji.
