# Template — Duel Storyboard

The strongest demo there is: same input, two columns. A generic AI beside the operator. The generic one does the confident, shallow thing; the operator refuses and does the disciplined thing. One judge called it "the whole pitch in 30 seconds." This storyboards it for a slide, a landing embed, or a video segment.

## The setup
- **One input, shown once, centered.** [the real input — a company, a brief, a request]
- Two columns below it, same prompt, side by side.

## Left column — generic AI
- [the confident paragraph it writes] — fluent, plausible, and exactly the thing the operator refuses to do (fabricates, profiles, guesses).
- Label it plainly: "Generic AI."

## Right column — the operator
- [the disciplined move] — it refuses the shallow answer and does the real work: flags the gap, routes the decision, asks the one resolving question.
- Label it with the operator's name.

## The payoff line
- [one line under the two columns] — names the difference in plain words. The refusal *is* the product.

## Notes
- Keep both columns the **same length** — the contrast is the message; don't let the operator's column sprawl.
- Static works (a slide). Motion is better (type the input, both columns fill, the generic one keeps going while the operator stops). See `reference/tooling.md` for OD vs Remotion.
- The input must be **real and checkable** — a judge should be able to fire the same input and get the same split.
