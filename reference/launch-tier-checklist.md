# Launch Material Checklist

What "done" looks like for each material in the Gate 2 leverage order. Gate 2 decides which materials to build; this defines the bar each one clears before it counts as shipped. No hour estimates — done is done.

## Pitch blurb (the floor)
- [ ] Two to three sentences: what workflow it handles + what decisions it makes.
- [ ] Leads with the hook (Gate 3), not the methodology.
- [ ] A stranger understands what it is without opening the repo.

## README hook polish (the floor)
- [ ] First paragraph passes the 5-minute stranger test.
- [ ] Setup is numbered and copy-pasteable.
- [ ] A "what's in the folder" table, one job per file.

## Landing page — the hub
- [ ] One page. Hero line = the hook. One primary CTA (the repo).
- [ ] The three things it does, in plain language.
- [ ] A visible artifact: a sample exchange or the duel.
- [ ] **It's the hub:** the video and deck embed or link from here — one URL to share, not a scatter of links.

## 60-second script
- [ ] 0–10s: the hook / the problem. 10–45s: it working on one real input. 45–60s: where to get it.
- [ ] Shows a decision being made, not a feature tour.
- [ ] Ships with a **separate spoken VO script** — the words alone, in the builder's voice, ready to record. (See `templates/60s-script.md`.)

## Walkthrough + judge guide + founder story
- [ ] Walkthrough: one full session, start to finish.
- [ ] Judge guide: 3–5 falsifiable prompts, each stating the behavior to expect. (See `reference/judging-criteria.md`.)
- [ ] Founder story: one line on why this builder, this problem.

## Pitch deck
- [ ] 8 slides on the brand's archetypes. The **comparison** slide carries the duel; the **data** slide carries the ROI / decision math.
- [ ] One idea per slide. Mixed-weight headline — one emphasized phrase.
- [ ] Built through the design system (RDS or the supplied kit), not freehand. (See `reference/tooling.md`.)

## Animated flow
- [ ] Shows the operator's decision in motion — input → gates → verdict — not a logo animation.
- [ ] OD HTML if it's a lightweight embed for the hub; Remotion if it's a segment in the video.
- [ ] Degrades to a legible end-state still (for PDF / thumbnail).

## Live demo (bonus)
- [ ] A stranger can drive it without setup hand-holding.
- [ ] Fails safe: a bad input gets a graceful answer, not a stack trace.
- [ ] One canned happy-path input is suggested on screen, so the value lands in the first ten seconds.
